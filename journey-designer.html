📋 ジャーニー全体方針

BOLD SCALPは「本格スカルプケアを続けやすい価格で」という強みを軸に、30〜40代男性の「何を使えばいいかわからない」という不安を解消するコンテンツ戦略を展開する。SNSでは教育型コンテンツで信頼を構築し、EC・LINEで購買・リピートへ誘導するCRM連動型設計とする。競合のスカルプDが「高価格・本格派」で訴求しているのに対し、継続しやすさとわかりやすさで差別化ポジションを確立する。

PHASE 1 🔍 認知
目標：薄毛に悩む男性へのブランド認知獲得
チャネル：YouTube・X
KPI：YouTube登録者+500人/月、Xインプレ10万/月
コンテンツ戦略：「市販シャンプーでは落とせない皮脂の話」など教育動画をYouTubeで週1投稿。Xでは#スカルプケア #薄毛対策タグを活用した短文ノウハウを毎日配信。
  1. YouTube「スカルプケア入門」シリーズ（全8本）の台本作成・撮影スケジュール策定
  2. 薄毛系インフルエンサー3〜5名へのコラボ打診
  3. X投稿カレンダーを作成し1日1投稿を習慣化
PHASE 2 💡 興味・教育
目標：成分の優位性と継続効果への理解を深める
チャネル：Instagram・YouTube

コンテンツ戦略：「なぜ効くのか」成分解説シリーズをInstagramカルーセルとYouTubeで展開。皮膚科医・毛髪診断士とのコラボで専門性を訴求。Before/After投稿はリール形式で保存率最大化を狙う。

PHASE 3 🤔 検討・比較

📊 業界トレンドからの示唆

スカルプケア市場は2025年に向けて男性化粧品全体の拡大とともに成長中。「成分へのこだわり」「継続使用による変化」をSNSで発信するブランドが高い支持を獲得しており、YouTubeショート・TikTokでの短尺動画が30〜40代男性への認知獲得に効果的。D2Cブランドへの直接購買意欲も高まっており、SNS→EC導線の整備が最優先課題となる。

🗣️ 消費者インサイト（日英口コミ分析）

薄毛に悩む30〜40代男性は「効果への懐疑心」と「男性向けブランドの少なさへの不満」を強く持っている。女性向け感のないブランドであることを明示することが信頼獲得の起点となる。また「何から始めればいいかわからない」という情報過多による意思決定麻痺が購買障壁になっており、診断ツールやステップ形式のガイドコンテンツで選択肢を絞る設計が有効。

✨ バズ・成功事例からのヒント

美容・ヘルスケアDtoCブランドのSNSバズ事例を見ると、Before/After写真・成分の科学的解説・専門家監修コンテンツが購買転換において圧倒的な効果を発揮している。特に男性向けケアでは「頑張らない努力・継続しやすさ」を訴求したコンテンツやモニター追跡シリーズが共感・保存・シェアされやすい。InstagramリールとLINE登録の組み合わせで教育→CRM誘導を設計するブランドがLTV最大化に成功している。

𝕏 X実投稿データから見えるユーザーの本音

「スカルプシャンプー 効果ない」「続けたら変わった」系の投稿が高エンゲージメント（❤️200〜800）。ユーザーは短期効果への不信感と長期継続への期待を同時に持っており、「3ヶ月後レポート」形式のUGCが最もバズりやすいパターン。成分の透明性を求める声も多く、「なぜ効くのか」コンテンツへの需要が高い。

⚔️ 競合（スカルプD）との差別化ポイント

スカルプDはX上で「高価格・本格派」として認知されているが、「続けにくい」「コスパが悪い」という不満が散見される。BOLD SCALPは継続しやすい価格帯＋わかりやすい成分説明を前面に出すことで「本格ケアを諦めかけていた層」の受け皿ポジションを獲得できる。YouTubeの成分解説コンテンツは競合が手薄なブルーオーシャン領域。

💰 LTV向上戦略

現状のリピート率30%・客単価5,800円・継続月数3ヶ月を前提とすると推定LTVは約17,400円。LINEステップ配信と残量アラートの導入で継続月数を6ヶ月に延ばし、定期購入転換率を高めることで目標LTV 34,800円（現状比2倍）を目指す。具体的には①購入後30日ステップ配信の自動化 ②定期購入プランへの誘導強化 ③3ヶ月継続特典の設計 ④紹介プログラムで新規獲得コスト削減の4施策を優先実施する。継続月数が6ヶ月を超えた顧客は「GOLD会員」として特別扱いし、解約率を抑制する。

🔍 検索精度のさらなる向上について

現在の実装はDuckDuckGo検索＋AI記事抽出を使用しています。以下のオプションを追加することで、検索リアル度をさらに高めることができます。

① Brave Search API（推奨）
Googleと同等の検索結果を取得可能。月2,000件まで無料。現在のコードへの影響が最小限で導入しやすい。
② Perplexity API
検索＋AI要約が一体化。現在の2ステップ処理が1回のAPI呼び出しで完結し、精度も最高水準。
③ claude -p ヘッドレス実行
Claude CodeのWebSearch/WebFetchツールをサブプロセスで呼び出し、Google検索ベースの高品質データを取得。
