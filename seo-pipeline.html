PROCESS

7工程の自動化フロー

Prompt A〜Eの5種類のプロンプトが7工程を自動実行します。各ステップの処理状況はリアルタイムで画面に表示されます。

1
Perplexityでリサーチ
キーワードをもとにPerplexityが最新情報を検索・収集。競合記事・トレンド・引用元URLを取得します。
Prompt A
2
リサーチ結果をフォームに貼り付け
Perplexityの出力結果をWebアプリのフォームに貼り付け。以降の工程は自動実行されます。
手動（ここだけ）
3
記事構成案の作成 + 執筆
SEOに最適な見出し構成を生成し、そのまま本文執筆まで一気に完了。文字数・トーン・構成はカスタマイズ可能。
Prompt B
4
ファクトチェック
執筆した記事内の事実情報（数字・固有名詞・統計）を再検証。不確かな情報を検出してフラグを立てます。
Prompt C
5
URL検証・置換
記事内の外部リンクURLにアクセスして生死確認。404などのリンク切れを自動検出し、代替URLに置換します。
Prompt C'
6
品質チェック
SEO観点・読みやすさ・構成の論理性・キーワード密度などを多角的に評価。改善提案も出力します。
Prompt D
7
内部リンク設計 → Google Docs保存
既存の自社記事一覧をもとに最適な内部リンクを設計。完成した記事をGoogle Docsに自動保存、またはダウンロード形式で出力。
Prompt E
